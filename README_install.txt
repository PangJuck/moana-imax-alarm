CGV IMAX 예매 오픈 알리미 — 서버 PC 설치 안내
================================================

무엇인가요?
  등록한 영화의 IMAX 예매가 용산아이파크몰 / 왕십리 / 천호 에서 열리면
  텔레그램으로 즉시 알려주는 프로그램입니다. (친구 여러 명 구독 가능)

준비물
  Windows PC, 한국 가정/사무실 인터넷(중요), 텔레그램 계정


==============================================
쉬운 방법: GUI 사용 (권장)
==============================================
파일을 직접 편집하지 않고 창에서 설정할 수 있습니다.
  1) 아래 [1] Python 설치, [2] 폴더 복사, [3] 봇 토큰 받기 까지는 동일하게 진행.
  2) gui.bat 더블클릭 -> 창이 열립니다.
  3) "텔레그램 봇 토큰" 칸에 토큰을 붙여넣고 "설정 저장".
  4) "감시 영화"에 제목을 입력하고 "추가" (예: 아바타). 추가 후 확인 결과가 로그에 표시됩니다.
  5) "감시 시작" 클릭. 휴대폰에서 봇에게 /start 를 보내면 구독 완료.
  6) 창을 켜둔 동안 감시됩니다. 현재 IMAX 상태는 "지금 상태 새로고침"으로 확인.
주의: GUI로 감시하는 동안에는 run.bat 이나 자동시작 서비스를 동시에 켜지 마세요(텔레그램 충돌).
PC 재부팅 후에도 창 없이 자동 실행을 원하면 아래 [7] 자동시작(서비스) 방식을 쓰고, 그때는 GUI는 닫아두세요.


==============================================
자세한 방법: 파일 직접 편집
==============================================

[1] Python 설치  (이미 설치돼 있으면 건너뜁니다)
    https://www.python.org/downloads/ 에서 최신 버전 설치.
    설치 첫 화면의 "Add python.exe to PATH" 를 반드시 체크하세요.

[2] 이 폴더를 서버 PC로 복사
    이 폴더(cgv-imax-alarm-server) 전체를 서버 PC의 원하는 곳에 둡니다.
    예: C:\cgv-imax-alarm

[3] 텔레그램 봇 만들기
    텔레그램에서 @BotFather 를 검색 → 대화 열기.
    /newbot 입력 → 봇 이름 입력 → 봇 아이디(반드시 bot 으로 끝남) 입력.
    BotFather 가 주는 토큰( 1234567:AAE... )을 복사합니다.

[4] 설정 입력
    config.json 을 메모장으로 열어
      "telegram_token": "여기에_봇토큰_붙여넣기"
    부분의 따옴표 안을 복사한 토큰으로 바꾸고 저장합니다.

    targets.json 에 감시할 영화 제목을 넣습니다(쉼표로 구분). 예:
      ["아바타", "F1 더 무비"]
    비워둬도 됩니다. 텔레그램에서 /add 제목 으로 나중에 추가할 수 있어요.

[5] 설치 및 점검
    setup.bat 더블클릭.
    requests 설치 후, 현재 감시 영화의 IMAX 상태를 1회 보여줍니다.
    오류 없이 결과가 나오면 정상입니다.

[6] 실행 테스트
    run.bat 더블클릭 → 검은 창에 로그가 흐르면 실행 중입니다.
    휴대폰 텔레그램에서 만든 봇을 찾아 /start 를 보냅니다.
    "구독되었습니다" 답장이 오면 연결 성공.
    확인 후 검은 창은 닫아도 됩니다([7]에서 자동 실행 등록).

[7] 자동 시작 등록  (PC 재부팅돼도 자동 실행 + 절전 방지)
    install_autostart.bat 더블클릭.
    이후 PC가 켜지고 로그인되면 창 없이 백그라운드로 자동 실행됩니다.
    해제는 uninstall_autostart.bat.


친구 구독 방법
    봇 링크(또는 봇 아이디)를 친구에게 알려주고, 각자 봇에게 /start 를
    누르면 자동으로 구독자에 등록됩니다. 모두에게 동시에 알림이 갑니다.

텔레그램 명령
    /list        감시 중인 영화 목록
    /add 제목     감시 영화 추가   (예: /add 아바타)
    /del 제목     감시 영화 제거
    /stop        내 구독 해지
    /help        도움말

참고 사항
    실행 로그는 bot.log 에 쌓입니다(용량이 커지면 삭제해도 됩니다).
    서버 PC는 한국 일반 인터넷이어야 CGV가 정상 응답합니다(해외/일부 클라우드 IP는 차단될 수 있음).
    토큰이 외부에 노출되면 BotFather 에서 /revoke 후 새 토큰으로 교체하세요.
    감시 주기는 config.json 의 poll_interval_sec(초). 기본 120초(2분).
